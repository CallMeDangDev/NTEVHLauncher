# NTEVHLauncher
> Fork of Wuthering Waves, made for NTE

Launcher cài đặt bản Việt Hoá **Neverness to Everness**.

## Cách dùng

1. Mở `NTEVHLauncher.exe`
2. Lần đầu mở, chọn hệ điều hành đang dùng (**Windows** hoặc **Linux / Wine**)
3. Launcher sẽ tự tìm thư mục game; nếu không tìm thấy, nhấn menu **⋯ → Thư mục game** để chọn thủ công
4. Nhấn **Cài đặt Việt Hoá** và chờ tải xong
5. Nhấn **Chơi Game** để vào game

## Tính năng

- Cài đặt / cập nhật / gỡ bản Việt Hoá tự động
- Hỗ trợ cả **Windows** (`netbios.dll`) và **Linux / Wine** (`version.dll`)
- Tự động nhận diện đường dẫn cài đặt game
- Cài font tuỳ chỉnh (thay thế font mặc định của bản Việt Hoá)
- Khởi chạy game trực tiếp từ launcher
- Tự cập nhật Launcher lên phiên bản mới nhất
- Chạy lại với quyền Admin nếu cần (**⋯ → Khởi động với Admin**)

## Yêu cầu

- Windows 10/11
- [WebView2 Runtime](https://developer.microsoft.com/en-us/microsoft-edge/webview2/) (thường đã có sẵn trên Windows 11)
